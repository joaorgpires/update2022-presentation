Análise Estática do Código

Um exemplo
							
								#include <string>
								#include <utility>
								#include <iostream>
								
								struct Player {
									int id;
									std::string ign;
									std::pair<float, float> pos;
									Player(int id, const std::string& ign, std::pair<float, float> pos)
									: id(id)
									, ign(ign),
									pos(pos) {}
								};
								
								int main() {
									Player *players[20];
									char *game_name = new char[20];
									std::cin >> game_name;
									for (int i = 0; i < 20; i++) {
										players[i] = new Player(i, "Player" + std::to_string(i), std::pair<float, float>(0.0f + i, 0.0f));
									}
									
									for(int i=0; i<20; i++)
									{
										std::cout << "Player " << players[i]->id<<" has in game name "<<players[i]->ign << " and is currently in (" << players[i]->pos.first << "," << players[i]->pos.second << ")" << std::endl;
									}

									delete players;
									delete game_name;
								}

Formatação Automática do Código

Formatação Automática do Código

Mais utilizado é o Clang-Format

Ferramenta desenvolvida e mantida pela llvm

O projeto LLVM começou em 2000 na Universidade de Illinois em Urbana-Champaign sob direção de Vikram Adve e Chris Lattner. Visava desenvolver uma infraestrutura de pesquisa para investigar técnicas de compilação dinâmica para linguagens estáticas e dinâmicas. A Apple Inc. contratou Lattner que formou um grupo de desenvolvedores no intuito de trabalhar no LLVM para sistemas próprios;[3] atualmente, o LLVM é parte integral das ferramentas de desenvolvimento para macOS e iOS.

								#include <iostream>
								#include <string>
								#include <utility>
								
								struct Player {
									int id;
									std::string ign;
									std::pair<float, float> pos;
									Player();
									Player(int id, const std::string& ign, std::pair<float, float> pos) : id(id), ign(ign), pos(pos) {}
								};
								
								int main() {
									Player* players[20];
									char* game_name = new char[20];

									for (int i = 0; i < 20; i++) {

									std::cout << game_name << std::endl;
